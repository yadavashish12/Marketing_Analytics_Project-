# Marketing Analytics Project

## Overview
This project analyzes customer journeys, social media engagement, and product reviews to uncover insights into customer behavior and marketing performance.

## Tools & Technologies
- SQL – Data cleaning and transformation
- Python (NLTK) – Sentiment analysis on customer reviews
- Power BI – Interactive dashboards and data visualization

## Key Analysis
- Customer journey analysis to understand conversion patterns
- Social media engagement trends
- Sentiment analysis of product reviews
- Marketing performance insights using interactive dashboards

## Key Insights
- Identified trends in customer engagement across channels
- Analyzed sentiment patterns influencing product perception
- Highlighted conversion trends across the customer journey

## Dashboard
Power BI dashboards visualize engagement metrics, sentiment insights, and conversion trends
